# 【無料】木造住宅の耐震性を5分でセルフチェック！「耐震診断くん」の使い方と技術解説

## はじめに：なぜ今、耐震診断が重要なのか

「うちは大丈夫だろう」

そう思っていませんか？

日本は世界有数の地震大国です。阪神・淡路大震災（1995年）、東日本大震災（2011年）、熊本地震（2016年）、能登半島地震（2024年）...私たちは繰り返し、地震の脅威と向き合ってきました。

特に注目すべきは、**1981年（昭和56年）以前に建てられた木造住宅**です。この年に建築基準法の耐震基準が大幅に改正され、いわゆる「新耐震基準」が施行されました。つまり、それ以前の建物は現在の基準を満たしていない可能性が高いのです。

国土交通省の調査によると、日本には約2,600万戸の木造住宅があり、そのうち約900万戸が耐震性不足と推計されています。あなたの住まいは大丈夫ですか？

「でも、耐震診断って専門家に頼むと高いし、面倒くさそう...」

そんな声にお応えして開発したのが、**「耐震診断くん」**です。

## サービス概要：2つの診断モードで幅広いニーズに対応

「耐震診断くん」は、木造住宅の耐震性能を診断できる無料のWebアプリです。専門知識がなくても使える「簡易診断」と、建築士向けの「精密診断」の2つのモードを用意しています。

### 1. 簡易診断（10問・約5分）

「まずは自分の家がどのくらい危ないか知りたい」という方向けのスクリーニングツールです。

**特徴：**
- 専門知識は一切不要
- 10個の質問に答えるだけ
- 所要時間は約5分
- 日本建築防災協会の「誰でもできるわが家の耐震診断」に準拠

**診断項目の例：**
- 建築年（1981年6月以降かどうか）
- 過去の災害被害の有無
- 増築の有無と適法性
- 建物の傷み具合（シロアリ・腐食など）
- 平面形状（整形か不整形か）
- 壁の配置バランス
- 屋根の重さと壁量のバランス
- 基礎の種類

**結果の見方：**
- **10点（満点）**：ひとまず安心。念のため専門家に診てもらいましょう
- **8〜9点**：少し心配な点があります。専門家への相談をおすすめします
- **7点以下**：心配な点が多いです。できるだけ早く専門家に診てもらいましょう

### 2. 精密診断（一般診断法準拠）

建築士や専門家が使用することを想定した、本格的な診断ツールです。

**特徴：**
- 2012年改訂版「木造住宅の耐震診断と補強方法」の精密診断法1に準拠
- **上部構造評点（Iw）** を算出
- 壁量計算・偏心率・劣化度を総合的に評価
- 補強計画と概算費用を自動提案

**診断の流れ：**
1. **建物情報入力**：階数、床面積、屋根材、地域区分など
2. **床・階の形状入力**：各階の縦横寸法
3. **壁の仕様入力**：壁の位置、種類、長さ、接合部仕様
4. **基礎・劣化度評価**：基礎形式、各部の劣化状態
5. **結果表示**：Iw値、方向別評価、補強提案

**評価基準：**
- **Iw >= 1.5**：倒壊しない
- **1.0 <= Iw < 1.5**：一応倒壊しない
- **0.7 <= Iw < 1.0**：倒壊する可能性がある
- **Iw < 0.7**：倒壊する可能性が高い

診断結果は**PDFレポート**としてダウンロードでき、過去の大地震との比較や、今後のロードマップも提示します。

## 技術的な特徴：モダンなWeb技術で安全・快適に

ここからは、開発者・エンジニアの方向けの技術解説です。

```

```

### 完全クライアントサイド計算：プライバシーを最優先

「耐震診断くん」の最大の特徴は、**すべての計算がブラウザ内で完結する**ことです。

通常のWebアプリでは、ユーザーが入力したデータをサーバーに送信し、サーバー側で計算を行います。しかし、耐震診断には住所や建物の詳細情報など、センシティブなデータが含まれます。

そこで私たちは、以下の設計方針を採用しました：

1. **入力データはサーバーに送信しない**
2. **計算ロジックはすべてTypeScriptで実装し、ブラウザで実行**
3. **PDFレポートもクライアントサイドで生成**

これにより、ユーザーのプライバシーを完全に保護しています。「自分の家の情報が漏れるかも」という心配は無用です。

### 計算ロジックの構成

精密診断の計算は、以下のモジュールで構成されています：

```typescript
src/lib/calc/
├── upper-structure-score.ts  // Iw値（上部構造評点）計算のメイン
├── wall-strength.ts          // 壁量・壁耐力計算
├── required-capacity.ts      // 必要耐力計算
├── eccentricity.ts           // 偏心率計算
├── deterioration.ts          // 劣化度係数計算
└── reinforcement.ts          // 補強提案生成
```

**上部構造評点（Iw）の計算式：**

```
Iw = edQu / Qr

edQu = Qu × eKfl × dK
  - Qu: 保有耐力（壁耐力の合計）
  - eKfl: 偏心率による低減係数
  - dK: 劣化低減係数
Qr: 必要耐力
```

各係数は、2012年改訂版の精密診断法1に準拠した定数テーブルから参照しています。

### PWA対応：オフラインでも使える

「耐震診断くん」は**PWA（Progressive Web App）** に対応しています。

- スマートフォンのホーム画面に追加可能
- オフラインでも基本機能が利用可能
- ネイティブアプリのような使用感

現地調査中にネットワークが不安定な場所でも、診断を継続できます。

### Zustandによる状態管理

診断は複数のステップに分かれているため、入力データを適切に管理する必要があります。Zustandを採用した理由は：

- **シンプルなAPI**：Reduxのような煩雑なボイラープレートが不要
- **軽量**：バンドルサイズへの影響が最小限
- **TypeScriptとの親和性**：型安全な状態管理が容易

## 使い方ガイド

### 簡易診断の場合

1. トップページで「簡易診断を始める」をクリック
2. 10個の質問に順番に回答（選択式）
3. 結果画面でスコアと判定を確認
4. 必要に応じてPDFレポートをダウンロード

### 精密診断の場合

1. トップページで「精密診断を始める」をクリック
2. 以下の情報を順番に入力：
   - 建物基本情報（階数、床面積、屋根材など）
   - 各階の床形状
   - 壁の仕様（位置、種類、接合部）
   - 基礎形式、劣化度
3. 「診断実行」で結果を表示
4. 補強提案と概算費用を確認
5. PDFレポートをダウンロード

## 開発の背景・想い

このアプリを開発したきっかけは、「耐震診断のハードルを下げたい」という想いでした。

専門家による正式な耐震診断は、一般的に10〜30万円程度の費用がかかります。もちろん、それには相応の価値がありますが、「まず自分の家がどのくらい危険なのか」を知るための入り口としては、ハードルが高すぎます。

「無料で、誰でも、今すぐ」耐震性をチェックできるツールがあれば、より多くの人が自宅の安全性に意識を向けるきっかけになるのではないか。

そんな思いから、「耐震診断くん」は生まれました。

もちろん、このツールは正式な耐震診断の代わりにはなりません。あくまで「スクリーニング」や「専門家に相談する前の事前確認」として使っていただきたいと考えています。
